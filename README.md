# The Esoteric Order Issues

Public issue tracker for The Esoteric Order of the Old Ones and Cthulhu
Cultists.

Use this repo to report bugs, broken levels, puzzle validation problems,
continue-code issues, audio problems, leaderboard problems, confusing copy, or
accessibility issues.

Please do not post private account details, secrets, tokens, or screenshots
that expose personal information.

Playable site: https://cthulhucultists.com
